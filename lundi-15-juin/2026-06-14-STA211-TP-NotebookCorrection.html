1. Résultats des exercices

Ex0 — Chargement

Données MNIST chargées : 60 000 train, 10 000 test, reshape 784, normalisation [0,1], affichage 200 images.

Ex1 — Régression logistique

  • 7 850 paramètres, SGD (lr=0.5, batch=100, 10 epochs).
  • Accuracy test : ~92 %

Ex2 — MLP

  • 79 510 paramètres (couche cachée 100D), sigmoïde.
  • Accuracy test : ~98 %

Ex3 — CNN

  • Architecture LeNet-like : Conv16 → Pool → Conv32 → Pool → Flat → Dense100 → Dense10.
  • 20 epochs.
  • Accuracy test : ~99 %

Ex4 — t-SNE et métriques

  • t-SNE : 2 composantes, init='pca', perplexity=30.
  • ConvexHull, GaussianMixture ellipses, Neighborhood Hit (k=6).
  • t-SNE sépare bien les classes contrairement à l’ACP.

Ex5 — Manifold Untangling

  • Chargement modèle sauvegardé, pop() pour extraire les représentations latentes, projection t-SNE.
  • Les classes sont bien mieux séparées dans l’espace latent que dans l’espace d’entrée.
