1. Deep Learning avec Keras

1.1 Exercice 0 — Chargement des données MNIST

MNIST : 60 000 images train, 10 000 test, $28\times28$, 10 classes (chiffres 0–9).

from keras.datasets import mnist (X_train, y_train), (X_test, y_test) = mnist.load_data() X_train = X_train.reshape(60000, 784).astype('float32') / 255 X_test = X_test.reshape(10000, 784).astype('float32') / 255

Afficher les 200 premières images avec Matplotlib.

1.2 Exercice 1 — Régression logistique

$p(\hat{y}_{c,i}|\mathbf{x}_i) = \frac{e^{\langle\mathbf{x}_i;\mathbf{w}_c\rangle + b_c}}{\sum_{c'} e^{\langle\mathbf{x}_i;\mathbf{w}_{c'}\rangle + b_{c'}}}$

Nombre de paramètres : $784\times 10 + 10 = 7\,850$.

model = Sequential() model.add(Dense(10, input_dim=784, name='fc1')) model.add(Activation('softmax'))

One-hot encoding :

from keras.utils import np_utils Y_train = np_utils.to_categorical(y_train, 10)
$\mathcal{L}_{\mathbf{W},\mathbf{b}}(\mathcal{D}) = -\frac{1}{N}\sum_{i=1}^N \log(\hat{y}_{c^*,i})$

Convexité : oui, LR + cross-entropy = problème convexe ⇒ minimum global assuré.

sgd = SGD(learning_rate=0.5) model.compile(loss='categorical_crossentropy', optimizer=sgd, metrics=['accuracy']) model.fit(X_train, Y_train, batch_size=100, epochs=10, verbose=1) scores = model.evaluate(X_test, Y_test, verbose=0)

Résultat attendu : $\sim$92 % accuracy test.

1.3 Exercice 2 — Perceptron multi-couches (MLP)

Une couche cachée $L=100$, sigmoïde, softmax en sortie.

Nombre de paramètres : $784\times 100 + 100 + 100\times 10 + 10 = 79\,510$.

model = Sequential() model.add(Dense(100, input_dim=784, name='fc1')) model.add(Activation('sigmoid')) model.add(Dense(10, name='fc2')) model.add(Activation('softmax'))

Non-convexité : MLP + sigmoïde = problème non convexe. Pas de garantie de minimum global.

Résultat attendu : $\sim$98 % accuracy test.

1.4 Exercice 3 — Réseau de neurones convolutif (CNN)

Reformatage : $28\times28\times1$.

X_train = X_train.reshape(-1, 28, 28, 1) X_test = X_test.reshape(-1, 28, 28, 1)

Architecture LeNet-like :

  1. Conv2D(16, $5\times5$, ReLU) + MaxPooling($2\times2$)
  2. Conv2D(32, $5\times5$, ReLU) + MaxPooling($2\times2$)
  3. Flatten + Dense(100, sigmoïde) + Dense(10, softmax)
model = Sequential() model.add(Conv2D(16, kernel_size=(5,5), activation='relu', input_shape=(28,28,1))) model.add(MaxPooling2D(pool_size=(2,2))) model.add(Conv2D(32, (5,5), activation='relu')) model.add(MaxPooling2D(pool_size=(2,2))) model.add(Flatten()) model.add(Dense(100, activation='sigmoid')) model.add(Dense(10, activation='softmax'))

Résultat attendu : $\sim$99 % accuracy test.

Sauvegarde du modèle

2. Deep Learning et Manifold Untangling

2.1 Exercice 4 — Visualisation avec t-SNE

t-SNE (t-Distributed Stochastic Neighbor Embedding) : réduction non linéaire en 2D qui préserve les proximités entre points.

from sklearn.manifold import TSNE tsne = TSNE(n_components=2, init='pca', perplexity=30, verbose=2) X_r = tsne.fit_transform(X_test[:1000])

Trois métriques de séparation

MétriqueDescription
Enveloppe convexe (ConvexHull)Périmètre minimum contenant tous les points d’une classe
Ellipse de meilleure approximationAjustement d’une gaussienne 2D par classe (GaussianMixture)
Neighborhood Hit (NH)Proportion des $k$ plus proches voisins de la même classe, moyennée sur toute la base
def neighboring_hit(points, labels, k=6): nbrs = NearestNeighbors(n_neighbors=k+1, algorithm='ball_tree').fit(points) distances, indices = nbrs.kneighbors(points) txs = 0.0 for i in range(len(points)): tx = sum(1 for j in range(1, k+1) if labels[indices[i,j]] == labels[i]) / k txs += tx return txs / len(points)
