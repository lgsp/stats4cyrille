1. Installation sous R

install.packages("keras")
library(keras)
install_keras()  # installe Keras + TensorFlow (backend)

install_keras() installe à la fois le cœur de Keras et le moteur TensorFlow. Keras est un front-end ; TensorFlow effectue les calculs.

2. Données

Problème de discrimination binaire dans le plan. 2000 observations, 2 variables X1, X2. Frontière parabolique :

Si 0.1 × X2 > X12 ⇒ pos, sinon neg

D <- read.table("artificial2d_data2.txt", sep="\t", header=T, dec=".")
y <- ifelse(D$Y=="pos", 1, 0)
set.seed(100)
idTrain <- sample(1:nrow(D), 1500, replace=F)
XTrain <- as.matrix(D[idTrain, 1:2])
yTrain <- y[idTrain]
XTest  <- as.matrix(D[-idTrain, 1:2])
yTest  <- y[-idTrain]

1500 observations en apprentissage, 500 en test.

3. Perceptron simple

Architecture

modelSimple <- keras_model_sequential()
modelSimple %>%
  layer_dense(units=1, input_shape=c(2), activation="sigmoid")
  • 1 neurone de sortie (sigmoïde), biais activé.
  • 3 paramètres : a0 + a1X1 + a2X2.

Compilation et apprentissage

modelSimple %>% compile(
  loss="binary_crossentropy",
  optimizer="adam",
  metrics="accuracy"
)
modelSimple %>% fit(x=XTrain, y=yTrain, epochs=150, batch_size=10)

Poids estimés : a0 = −1.283, a1 = 0.256, a2 = 1.947.

Performances

Test accuracy : 67.8 %

4. Perceptron multicouche (MLP)

Architecture

modelMc <- keras_model_sequential()
modelMc %>%
  layer_dense(units=3, input_shape=c(2), activation="sigmoid") %>%
  layer_dense(units=1, activation="sigmoid")
  • 1 couche cachée (3 neurones, sigmoïde).
  • 1 couche sortie (1 neurone, sigmoïde).
  • 13 paramètres : 9 (entrée → cachée) + 4 (cachée → sortie).

Performances

Test accuracy : 97.4 %

Net gain par rapport au perceptron simple (67.8 %). La couche cachée capture la frontière parabolique non linéaire.
