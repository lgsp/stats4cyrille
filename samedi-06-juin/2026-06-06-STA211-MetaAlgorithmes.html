1. Présentation

Les méta-algorithmes (méthodes d'ensemble) combinent plusieurs modèles faibles (weak learners) pour obtenir un prédicteur plus stable et plus performant.

Principe : on entraîne \(B\) modèles de base \(\hat{f}_1, \dots, \hat{f}_B\) sur des versions perturbées des données, puis on agrège leurs prédictions.

\( \text{Régression : } \displaystyle \hat{f}_{\text{ens}}(\mathbf{x}) = \frac{1}{B}\sum_{b=1}^{B} \hat{f}_b(\mathbf{x}) \qquad \text{Classification : } \displaystyle \hat{f}_{\text{ens}}(\mathbf{x}) = \arg\max_k \sum_{b=1}^{B} \mathbf{1}_{\{\hat{f}_b(\mathbf{x}) = k\}} \)

Forêts aléatoiresBootstrap + sous-échantillonnage variablesArbres profonds
BoostingPondération des individusArbres peu profonds (souches)
StackingPartitionnement des donnéesModèles variés

2. Bagging

Principe (Breiman, 1996) : générer \(B\) échantillons bootstrap, entraîner un modèle sur chacun, agréger par moyenne/vote.

2.1 Propriétés

  • Réduction de la variance : pour des modèles identiquement distribués avec variance \(\sigma^2\) et corrélation \(\rho\),
\( \displaystyle \mathbb{V}\!\left[\frac{1}{B}\sum_{b=1}^B \hat{f}_b\right] = \rho\sigma^2 + \frac{1-\rho}{B}\sigma^2 \)
  • Quand \(B \to \infty\), variance tend vers \(\rho\sigma^2\).
  • Efficace pour modèles instables (forte variance, faible biais).
  • Chaque bootstrap utilise \( \approx 63{,}2\% \) des données ; les \(36{,}8\%\) restants sont OOB.

2.2 Erreur OOB

\( \displaystyle \text{Err}_{\text{OOB}} = \frac{1}{n}\sum_{i=1}^n \mathcal{L}\!\left(y_i,\; \frac{1}{|\mathcal{B}_i|}\sum_{b \in \mathcal{B}_i} \hat{f}_b(\mathbf{x}_i)\right) \)

où \(\mathcal{B}_i\) = arbres pour lesquels l'individu \(i\) est OOB.

3. Forêts aléatoires

Random Forests (Breiman, 2001) : bagging + sous-échantillonnage des variables. À chaque nœud, on tire \(m\) variables parmi \(p\).

3.1 Hyperparamètres

ParamètreValeur typiqueEffet
\(B\) (nombre d'arbres)100-1000Plus grand → plus stable
\(m\) (variables/nœud)\(\sqrt{p}\) (classif.) / \(p/3\) (régr.)Petit → décorrélation
Profondeur maxIllimitéeArbres profonds = faible biais
Taille min. feuilles1–5Évite surajustement

3.2 Importance des variables

\( \displaystyle \text{Imp}(X_j) = \sum_{\text{nœuds } t \text{ divisés par } X_j} \Delta_t \)

où \(\Delta_t\) = réduction d'hétérogénéité (Gini ou RSS) au nœud \(t\).

4. Boosting

Approche séquentielle : chaque nouveau modèle corrige les erreurs du précédent.

4.1 AdaBoost

Freund & Schapire, 1997. Pondère les individus par leur difficulté :

  1. Initialiser \(w_i^{(1)} = 1/n\).
  2. Pour \(b = 1, \dots, B\) :
    1. Entraîner \(\hat{f}_b\) sur données pondérées par \(\mathbf{w}^{(b)}\).
    2. Erreur pondérée : \(\displaystyle \varepsilon_b = \frac{\sum_i w_i^{(b)} \mathbf{1}_{\{\hat{f}_b(\mathbf{x}_i) \neq y_i\}}}{\sum_i w_i^{(b)}}\)
    3. Poids du classifieur : \(\displaystyle \alpha_b = \frac12\log\!\left(\frac{1-\varepsilon_b}{\varepsilon_b}\right)\)
    4. Mettre à jour : \(w_i^{(b+1)} = w_i^{(b)} \exp[-\alpha_b y_i \hat{f}_b(\mathbf{x}_i)]\), normaliser.
  3. Prédiction : \(\displaystyle \hat{f}(\mathbf{x}) = \text{signe}\!\left(\sum_{b=1}^B \alpha_b \hat{f}_b(\mathbf{x})\right)\)

4.2 Gradient Boosting

Friedman, 2001. Généralise AdaBoost à toute fonction de perte différentiable :

  1. Initialiser \(\displaystyle \hat{f}^{(0)}(\mathbf{x}) = \arg\min_\gamma \sum_i \mathcal{L}(y_i, \gamma)\).
  2. Pour \(b = 1, \dots, B\) :
    1. Résidus (gradient négatif) : \(\displaystyle r_i^{(b)} = -\frac{\partial \mathcal{L}(y_i, f(\mathbf{x}_i))}{\partial f(\mathbf{x}_i)}\big|_{f = \hat{f}^{(b-1)}}\)
    2. Ajuster un arbre \(\hat{f}_b\) aux résidus \(\{(\mathbf{x}_i, r_i^{(b)})\}\).
    3. Mettre à jour : \(\hat{f}^{(b)}(\mathbf{x}) = \hat{f}^{(b-1)}(\mathbf{x}) + \eta \cdot \hat{f}_b(\mathbf{x})\).

Hyperparamètres : \(B\) (arbres), \(\eta \in [0{,}01; 0{,}3]\) (taux d'apprentissage), profondeur 1–3.

5. Stacking

Stacked Generalization (Wolpert, 1992) : combine des modèles de nature différente via un méta-modèle.

Niveau 0 (base) : \(K\) modèles \(\hat{f}_1, \dots, \hat{f}_K\) (arbre, régression logistique, SVM, kNN…).
Niveau 1 (méta) : modèle \(\hat{g}\) qui apprend à combiner les prédictions du niveau 0.

Procédure (validation croisée pour éviter le surajustement) :

  1. Diviser les données en \(V\) plis.
  2. Pour chaque pli \(v\) : entraîner les \(K\) modèles sur les \(V-1\) autres plis, prédire sur le pli \(v\).
  3. Matrice \(\mathbf{Z}\) (\(n \times K\)) des prédictions hors-échantillon.
  4. Entraîner \(\hat{g}\) sur \((\mathbf{Z}, \mathbf{y})\).

Le méta-modèle est souvent simple : régression logistique, régression linéaire, petit arbre.

6. Code Python

from sklearn.ensemble import ( BaggingRegressor, RandomForestRegressor, GradientBoostingRegressor, StackingRegressor ) from sklearn.tree import DecisionTreeRegressor from sklearn.linear_model import LinearRegression from sklearn.svm import SVR from sklearn.model_selection import train_test_split from sklearn.metrics import mean_squared_error import numpy as np # Donnees synthetiques np.random.seed(42) X = np.random.randn(200, 5) y = X[:, 0] + 2 * X[:, 1] + np.sin(X[:, 2]) + 0.5 * np.random.randn(200) X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2) # Bagging bag = BaggingRegressor( estimator=DecisionTreeRegressor(max_depth=None), n_estimators=100 ).fit(X_train, y_train) print(f"Bagging RMSE: {mean_squared_error(y_test, bag.predict(X_test), squared=False):.3f}") # Foret aleatoire rf = RandomForestRegressor( n_estimators=200, max_features='sqrt' ).fit(X_train, y_train) print(f"RF RMSE: {mean_squared_error(y_test, rf.predict(X_test), squared=False):.3f}") # Gradient Boosting gb = GradientBoostingRegressor( n_estimators=200, learning_rate=0.1, max_depth=3 ).fit(X_train, y_train) print(f"GB RMSE: {mean_squared_error(y_test, gb.predict(X_test), squared=False):.3f}") # Stacking stack = StackingRegressor( estimators=[ ('rf', RandomForestRegressor(n_estimators=100)), ('svr', SVR(kernel='rbf')), ('lr', LinearRegression()) ], final_estimator=LinearRegression(), cv=5 ).fit(X_train, y_train) print(f"Stacking RMSE: {mean_squared_error(y_test, stack.predict(X_test), squared=False):.3f}")

7. Code R

library(rpart) library(randomForest) library(gbm) library(caret) # Donnees set.seed(42) n <- 200 X <- matrix(rnorm(n * 5), n, 5) y <- X[,1] + 2*X[,2] + sin(X[,3]) + 0.5*rnorm(n) dat <- data.frame(y, X) # Foret aleatoire rf <- randomForest(y ~ ., data = dat, ntree = 200, importance = TRUE) print(rf) importance(rf) # Gradient Boosting gb <- gbm(y ~ ., data = dat, distribution = "gaussian", n.trees = 200, interaction.depth = 3, shrinkage = 0.1, cv.folds = 5) best <- gb.perf(gb, method = "cv") summary(gb, n.trees = best)
