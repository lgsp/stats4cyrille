1. Objectifs et contexte

Quand les variables sont nombreuses, il est parfois très utile de les regrouper en groupes homogènes. Ceci peut servir à :

  • Analyser la multicolinéarité entre les variables.
  • Réduire le nombre de variables en leur substituant un petit nombre de variables latentes synthétiques représentatives de chaque classe.

On distingue trois stratégies de partitionnement :

  1. Les méthodes hiérarchiques ascendantes (CAH).
  2. Les méthodes hiérarchiques descendantes.
  3. Les méthodes de partitionnement direct.

2. Méthodes hiérarchiques

2.1 Méthode ascendante (CAH)

Les groupements se font par agglomération progressive des variables deux à deux. On réunit les variables les plus proches au sens d’une dissimilarité, en utilisant un critère d’agrégation.

La complexité est en $O(p^3)$ où $p$ est le nombre de variables.

Indices de similarité/dissimilarité

Le choix de l’indice dépend de la nature des variables.

Variables quantitatives : le coefficient de corrélation linéaire au carré $r_{jj'}^2$ est l’indice naturel de similarité. La dissimilarité associée $d = 1 - r^2$ est une distance euclidienne.

Variables qualitatives : les indices les plus classiques sont le $\chi^2$ de contingence, le Tschuprow ($T$) et le Cramer ($V$).

Variables binaires : plusieurs indices définis à partir du tableau de contingence $2\times2$ ($N_{11}, N_{10}, N_{01}, N_{00}$) :

IndiceFormule
Sokal-Michener$\frac{N_{11}+N_{00}}{N_{11}+N_{01}+N_{10}+N_{00}}$
Jaccard$\frac{N_{11}}{N_{11}+N_{01}+N_{10}}$
Russel-Rao$\frac{N_{11}}{N_{11}+N_{01}+N_{10}+N_{00}}$
Ochiai$\frac{N_{11}}{\sqrt{(N_{11}+N_{01})(N_{11}+N_{10})}}$
Dice$\frac{2N_{11}}{2N_{11}+N_{01}+N_{10}}$
Rogers-Tanimoto$\frac{N_{11}+N_{00}}{N_{11}+2(N_{01}+N_{10})+N_{00}}$
Kulzinsky$\frac{N_{11}}{N_{01}+N_{10}}$
Remarque : Les mesures de dissimilarité associées sont toutes euclidiennes (Fichet & le Calve, 1984).

Variables mixtes : deux méthodes proposées par Hummel, Edelmann & Kopp-Schneider (2017) :

  • CluMix-ama : similarité basée sur la valeur absolue du $\gamma$ de Goodman-Kruskal pour les couples mixtes.
  • CluMix-dcor : similarité basée sur une distance de Gower entre individus.

Ces méthodes sont implémentées dans le package R CluMix.

Critères d’agrégation

CritèreDistance entre groupes $A$ et $B$
Saut minimum$\min_{i \in A, j \in B} d(i,j)$
Saut maximum$\max_{i \in A, j \in B} d(i,j)$
Lien moyen$\frac{1}{|A||B|} \sum_{i \in A} \sum_{j \in B} d(i,j)$
Ward$\frac{|A||B|}{|A|+|B|} \| \mu_A - \mu_B \|^2$

2.2 Méthode descendante (VARCLUS)

Procède par dichotomies successives pour construire un arbre hiérarchique. Méthode VARCLUS (SAS) :

  1. Réaliser une ACP des variables.
  2. Si la seconde valeur propre $> 1$, retenir les deux premières composantes.
  3. Affecter chaque variable à la composante avec laquelle elle est le plus corrélée.
  4. Répéter sur chaque groupe tant que la seconde valeur propre $> 1$.
À retenir : VARCLUS est différente de ClustOfVar. Il existe une implémentation Python (variable-clustering) mais pas d’implémentation R standard.

3. Partitionnement direct

3.1 Méthode CLV (Vigneau & Qannari, 2003)

Destinée aux variables quantitatives. Approche similaire aux $K$-means, maximisant un critère de colinéarité intra-classe :

$\displaystyle \sum_{k=1}^K \sum_{j=1}^p \delta_{k,j} \; \text{cov}^2(x_j, u_k) \quad\text{sous contrainte}\quad \|u_k\| = 1$

où $\delta_{k,j}=1$ si la variable $j$ est dans la classe $k$, et $u_k$ est la première composante principale de la classe $k$.

Algorithme :

  1. Initialiser les classes.
  2. Étape d’affectation : calculer $\delta_{k,j}$ (affecter chaque variable à la classe dont le centroïde est le plus proche).
  3. Étape de mise à jour : recalculer $u_k$ (1ère CP de chaque classe).
  4. Répéter jusqu’à convergence.

3.2 ClustOfVar (Chavent et al., 2012)

Généralisation de CLV aux variables qualitatives ou mixtes :

  • Utilise les composantes principales de l’ACM (qualitatives) ou de l’AFDM (mixtes) comme centroïde $u_k$.
  • Similarité mesurée par le carré du rapport de corrélation (qualitative) ou le carré de la corrélation (quantitative).
À retenir : Les algorithmes de partitionnement direct de variables restent sensibles à leur initialisation. Tester plusieurs initialisations et retenir celle optimisant le critère.

4. Application aux données German Credit

Jeu de données German Credit ($n=1000$, $p=21$, 7 quantitatives, 14 qualitatives). Objectif : classifier les variables mixtes.

4.1 Chargement et CAH avec ClustOfVar

library(CluMix); library(ClustOfVar) library(cluster); library(FactoMineR) don <- read.table( "https://archive.ics.uci.edu/ml/machine-learning- databases/statlog/german/german.data", sep = " ", stringsAsFactors = TRUE) var.factor <- which(sapply(don, class) == "factor") var.numeric <- which(sapply(don, class) %in% c("numeric", "integer")) set.seed(1234) res.cah.hclustvar <- hclustvar( X.quanti = don[,var.numeric], X.quali = don[,var.factor]) k_hclustvar <- 4 res.cutree <- cutreevar(res.cah.hclustvar, k = k_hclustvar)

4.2 CAH avec CluMix

# CluMix-ama res.dendro_ama <- dendro.variables(don) res.dendro_ama <- as.hclust(res.dendro_ama) k_ama <- 3; part.cah.CluMix_ama <- cutree(res.dendro_ama, k = k_ama) # CluMix-dcor res.dendro_dcor <- dendro.variables(don, method = "distcor") res.dendro_dcor <- as.hclust(res.dendro_dcor) k_dcor <- 4; part.cah.CluMix_dcor <- cutree(res.dendro_dcor, k = k_dcor)

4.3 Partitionnement avec ClustOfVar et PAM

res.kmeansvar <- kmeansvar( X.quanti = don[,var.numeric], X.quali = don[,var.factor], init = k_hclustvar, nstart = 100) # Meilleure initialisation via CAH ou tirage aléatoire myrandominit <- function(matdist, nb.centres){ random_centre <- sample(seq.int(ncol(don)), size = nb.centres) random_part <- apply(matdist[,random_centre], 1, which.min) return(random_part) } matsim <- PairApply(don, mixedVarSim, symmetric = TRUE) matdist <- as.matrix(sqrt(round(1 - matsim, 5))) random_init <- replicate(100, myrandominit(matdist, k_hclustvar), simplify = FALSE) list.init <- c(list(cah.CluMix_dcor = part.cah.CluMix_dcor, cah.hclustvar = part.cah.hclustvar), random_init) # PAM sur dissimilarités diss.CluMix_ama <- dist.variables(don) res.pam_ama <- pam(diss.CluMix_ama, diss = TRUE, k = k_ama)

4.4 Résultats : quatre groupes de variables

Une ACM sur le tableau des partitions met en évidence quatre groupes :

  1. Profil personnel : Length.of.current.employment, Sex.Marital.Status, Duration.in.Current.address, Age.years, Housing, No.of.dependents.
  2. Type de crédit et profil financier : Duration, Purpose, Credit.Amount, Instalment.per.cent, Guarantors, Job, Telephone, Foreign.Worker.
  3. Actifs du demandeur : Status, Savings account/bonds, Other.installment.plans, Creditability.
  4. Historique : History, No.of.Credits.at.this.Bank.
Observation : La variable Creditability (réponse) est liée au groupe des actifs, ce qui suggère une relation privilégiée avec ces variables.

5. Conclusion

La classification de variables vise à regrouper les variables en classes homogènes : les variables d’un même groupe sont fortement liées, celles de groupes différents sont faiblement liées.

Applications importantes :

  • Fuzzy forests : variante des forêts aléatoires qui classifie d’abord les variables corrélées, puis effectue une sélection backward au sein de chaque groupe (package R fuzzyforest).
  • Biclustering (co-clustering) : classification simultanée des individus et des variables pour identifier des groupes d’individus caractérisés par des sous-ensembles de variables.
