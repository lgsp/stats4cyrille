1. Présentation

Les arbres CART (Breiman et al., 1984) sont des méthodes supervisées non-paramétriques qui partitionnent l'espace des variables explicatives en régions et associent une prédiction simple à chaque région.

✅ Avantages

  • Interprétabilité (règles lisibles)
  • Gère variables mixtes (quantitatives + qualitatives)
  • Pas d'hypothèse sur la distribution

⚠️ Inconvénients

  • Forte variance (instabilité)
  • Biais de prédiction
  • Nécessite élagage pour généraliser
💡 Types : Régression (Y quantitative → moyenne) • Classification (Y qualitative → mode)

2. Architecture

Un arbre est composé de la racine (tous les individus), de nœuds internes (divisions), et de feuilles (régions terminales).

Prédiction : \( f(\mathbf{x}) = \sum_{m=1}^{M} \hat{y}_{R_m} \cdot \mathbb{1}_{\{\mathbf{x} \in R_m\}} \)
Régression : \(\hat{y}_{R_m} = \frac{1}{n_m}\sum_{i \in R_m} y_i\)  |  Classification : \(\hat{y}_{R_m} = \arg\max_k \hat{p}_{mk}\)

3. Algorithme de construction

3.1 Partitionnement récursif binaire

  1. Partir de la racine (tous les individus).
  2. Pour chaque nœud, trouver la variable \(X_j\) et le seuil \(s\) minimisant l'hétérogénéité.
  3. Diviser en deux fils selon \(X_j < s\) et \(X_j \ge s\).
  4. Répéter jusqu'à un critère d'arrêt.

3.2 Critères de division

📈 Régression (RSS)

Minimiser : \(\sum_{i: x_{ij} < s} (y_i - \bar{y}_g)^2 + \sum_{i: x_{ij} \ge s} (y_i - \bar{y}_d)^2\)

🏷️ Classification

Gini : \(G = 1 - \sum_k \hat{p}_{mk}^2\)
Entropie : \(D = -\sum_k \hat{p}_{mk} \log \hat{p}_{mk}\)

4. Élagage (pruning)

Un arbre complet surajuste. L'élagage par coût-complexité pénalise le nombre de feuilles :

Critère\(_\alpha(A)\) = Erreur(\(A\)) + \(\alpha \cdot |A|\)

où \(|A|\) = nombre de feuilles, \(\alpha \ge 0\) choisi par validation croisée.

🔑 \(\alpha = 0\) → arbre complet. \(\alpha\) grand → arbre simple. On sélectionne \(\alpha\) par 10-fold CV.

6. Code Python

from sklearn.tree import DecisionTreeRegressor, DecisionTreeClassifier
from sklearn.tree import plot_tree
from sklearn.model_selection import train_test_split
from sklearn.datasets import load_iris

# Arbre de classification (Iris)
iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.3, random_state=42)

tree = DecisionTreeClassifier(max_depth=4,
                               criterion='gini',
                               min_samples_leaf=5)
tree.fit(X_train, y_train)
y_pred = tree.predict(X_test)

# Importance des variables
for name, imp in zip(iris.feature_names, tree.feature_importances_):
    print(f"{name}: {imp:.3f}")

# Elagage par cout-complexite
path = tree.cost_complexity_pruning_path(X_train, y_train)
from sklearn.model_selection import GridSearchCV
grid = GridSearchCV(
    DecisionTreeClassifier(random_state=42),
    {'ccp_alpha': path.ccp_alphas}, cv=5)
grid.fit(X_train, y_train)
best = grid.best_estimator_
print(f"Meilleur alpha: {grid.best_params_['ccp_alpha']:.4f}")
print(f"Feuilles: {best.get_n_leaves()}")

7. Code R

library(rpart)
library(rpart.plot)

# Arbre de classification (Iris)
data(iris)
idx <- sample(1:nrow(iris), 0.7 * nrow(iris))
train <- iris[idx, ]
test  <- iris[-idx, ]

arbre <- rpart(Species ~ ., data = train,
              method = "class",
              control = rpart.control(minsplit = 5,
                                       maxdepth = 4,
                                       cp = 0.01))
rpart.plot(arbre, type = 2, extra = 104)

# Elagage par cp
best_cp <- arbre$cptable[which.min(arbre$cptable[, "xerror"]), "CP"]
arbre_elague <- prune(arbre, cp = best_cp)
rpart.plot(arbre_elague, type = 2, extra = 104)

# Arbre de regression
X <- seq(0, 10, length.out = 100)
y <- sin(X) + rnorm(100, sd = 0.1)
tree_reg <- rpart(y ~ X, method = "anova")
rpart.plot(tree_reg, type = 2)
