1. Présentation

Les cartes de Kohonen (SOM) sont des réseaux de neurones artificiels en apprentissage non supervisé qui projettent des données de grande dimension sur une grille 2D tout en préservant la topologie : des points voisins dans l'espace d'entrée restent voisins sur la carte.

🎯 Objectifs

  • Réduction de dimension (visualisation)
  • Clustering non supervisé
  • Conservation de la topologie

✨ Propriété clé

Deux points proches dans l'espace d'entrée sont représentés par des neurones proches (ou identiques) sur la carte.

2. Architecture

La carte est composée de deux couches :

  • Couche d'entrée : chaque neurone correspond à une variable (dimension d).
  • Couche de sortie (carte) : grille de neurones (souvent rectangulaire ou hexagonale). Chaque neurone j a un vecteur de poids wj ∈ ℝd.
\(\text{Dimension d'entrée : } d \to \text{Grille } n_x \times n_y \text{ neurones}\)

3. Algorithme

3.1 Principe

  1. Initialisation : poids aléatoires wj(0).
  2. Pour chaque individu x(i) :
    1. Trouver le BMU (Best Matching Unit) — neurone le plus proche.
    2. Mettre à jour le BMU et ses voisins.
  3. Répéter en diminuant α et σ.

3.2 Formules

\(\text{BMU : } c = \arg\min_j \|\mathbf{x} - \mathbf{w}_j\|\)
\(\mathbf{w}_j(t+1) = \mathbf{w}_j(t) + \alpha(t) \cdot h_{c,j}(t) \cdot (\mathbf{x} - \mathbf{w}_j(t))\)
\(h_{c,j} = \exp\!\big(-\|\mathbf{r}_c - \mathbf{r}_j\|^2 / 2\sigma^2\big)\)
\(\alpha(t) = \alpha_0 (1 - t/T),\; \sigma(t) = \sigma_0 (1 - t/T)\)
Note : Le BMU est mis à jour avec h = 1 (le plus fortement), ses voisins avec h < 1 (d'autant moins que la distance sur la grille est grande).

5. Code Python

5.1 Implémentation simple avec NumPy

import numpy as np
import matplotlib.pyplot as plt

class SOM:
    def __init__(self, grid_size, d, alpha_0=0.5, sigma_0=1.0):
        self.grid_size = grid_size
        n_neurons = grid_size[0] * grid_size[1]
        self.W = np.random.randn(n_neurons, d)
        self.alpha_0 = alpha_0
        self.sigma_0 = sigma_0
        self.positions = np.array([(i,j) for i in range(grid_size[0])
                                           for j in range(grid_size[1)])

    def bmu(self, x):
        dist = np.linalg.norm(self.W - x, axis=1)
        return np.argmin(dist)

    def fit(self, X, n_epochs=10):
        n = len(X)
        for epoch in range(n_epochs):
            alpha = self.alpha_0 * (1 - epoch/n_epochs)
            sigma = self.sigma_0 * (1 - epoch/n_epochs)
            np.random.shuffle(X)
            for x in X:
                c = self.bmu(x)
                for j in range(len(self.W)):
                    d_grid = np.linalg.norm(
                        self.positions[c] - self.positions[j])
                    h = np.exp(-d_grid**2 / (2 * sigma**2))
                    self.W[j] += alpha * h * (x - self.W[j])

    def predict(self, X):
        return np.array([self.bmu(x) for x in X])

5.2 Application sur Iris

from sklearn.datasets import load_iris
from minisom import MiniSom

iris = load_iris()
X = iris.data
X_norm = (X - X.mean(axis=0)) / X.std(axis=0)

# Modele SOM (grille 5x5)
som = MiniSom(5, 5, 4, sigma=1.0, learning_rate=0.5)
som.random_weights_init(X_norm)
som.train_random(X_norm, 500)

# U-Matrix
plt.imshow(som.distance_map(), cmap='bone_r')
plt.colorbar()
plt.title("U-Matrix — Iris")
plt.show()

6. Code R

library(kohonen)

data(iris)
X <- as.matrix(iris[, 1:4])
X_norm <- scale(X)

# Modele SOM
grille <- somgrid(xdim=5, ydim=5, topo="hexagonal")
som_model <- som(X_norm, grid=grille, rlen=500,
                alpha=c(0.5, 0.01), radius=c(2, 0.1))

# U-Matrix
plot(som_model, type="dist.neighbours",
     palette.name=topo.colors, main="U-Matrix")

# Projection des especes
especes <- as.integer(iris$Species)
plot(som_model, type="mapping", bg=rainbow(3)[especes],
     main="Projection des iris")

7. Visualisations

🗺️ U-Matrix

Distance moyenne entre chaque neurone et ses voisins. Zones claires = clusters, zones sombres = frontières.

📊 Effectifs (counts)

Nombre d'individus par neurone. Certains neurones morts peuvent indiquer un mauvais dimensionnement.

📐 Plans de composantes

Valeur d'une variable sur chaque neurone. Permet d'interpréter les clusters.

📈 Erreur de quantification (MQE)

MQE = (1/n) Σ ∥x(i) − wc(i)∥. Plus elle est faible, mieux la carte représente les données.

8. Résumé

ConceptÀ retenir
TypeApprentissage non supervisé
ObjectifRéduction de dimension + conservation topologie
BMUNeurone le plus proche (distance euclidienne)
Mise à jourwj ← wj + α · hc,j · (x − wj)
VoisinageGaussien : h = exp(−d²/2σ²)
Décroissanceα et σ diminuent avec le temps
BibliothèquesPython : minisom · R : kohonen
Points clés à retenir pour l'examen :
  1. Le BMU se trouve par distance euclidienne minimale.
  2. La mise à jour utilise un voisinage gaussien qui décroît.
  3. La carte préserve la topologie des données d'entrée.
  4. La U-Matrix visualise les distances entre neurones voisins (frontières de clusters).
