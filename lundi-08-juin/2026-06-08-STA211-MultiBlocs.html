1. Présentation

L'analyse multi-blocs regroupe des méthodes factorielles pour analyser simultanément plusieurs tableaux décrivant les mêmes individus. Chaque tableau est un bloc.

📊 Contexte

\(N\) tableaux \(\mathbf{X}_1 \ldots \mathbf{X}_N\), mêmes \(n\) individus, variables possiblement différentes (\(p_1 \ldots p_N\)).

🎯 Objectifs

  • Comparer les groupes de variables
  • Trouver un compromis commun
  • Visualiser l'évolution (trajectoires)
  • Réduire la dimension

2. Les 4 phases de l'analyse multi-blocs

  1. Interstructure : on résume chaque bloc par un objet (centre de gravité, matrice de produits scalaires…) et on les compare.
  2. Compromis : on cherche un espace commun maximisant l'inertie projetée de tous les nuages.
  3. Intrastructure : on analyse le compromis (cercles des corrélations, contributions…).
  4. Trajectoires : on projette les individus de chaque bloc dans le plan du compromis.

3. Méthodes

3.1 Double ACP (DACP)

Quand : mêmes variables, mêmes individus (ex. données temporelles).

Compromis : matrice \(\mathbf{V} = \sum_t \mathbf{V}_t\) (somme des matrices de variance-covariance).

3.2 STATIS

Quand : mêmes individus, variables différentes selon les blocs.

Interstructure : matrices de produits scalaires \(\mathbf{W}_t = \mathbf{X}_t \mathbf{M}_t \mathbf{X}_t^\top\) (dimension \(n \times n\)).

3.3 AFM (Analyse Factorielle Multiple)

Quand : mêmes individus, variables de types potentiellement différents. Chaque bloc est pondéré par l'inverse de sa première valeur propre.

💡 Résumé : DACP = ACP sur centres de gravité puis ACP sur la somme des covariances. STATIS = ACP sur matrices de produits scalaires. AFM = ACP pondérée par bloc.

5. Code Python

import numpy as np
import pandas as pd
from sklearn.decomposition import PCA

def dacp(list_of_dfs):
    n_blocs = len(list_of_dfs)
    p = list_of_dfs[0].shape[1]

    # Interstructure : ACP sur les centres de gravite
    centers = np.array([df.mean(axis=0).values for df in list_of_dfs])
    pca_inter = PCA().fit(centers)

    # Compromis : somme des matrices de covariance
    V_sum = np.zeros((p, p))
    for df in list_of_dfs:
        c = df - df.mean(axis=0)
        V_sum += c.T @ c / (len(df) - 1)
    V_sum /= n_blocs

    eigvals, eigvecs = np.linalg.eigh(V_sum)
    idx = np.argsort(eigvals)[::-1]
    eigvals, eigvecs = eigvals[idx], eigvecs[:, idx]

    return eigvals, eigvecs, pca_inter
